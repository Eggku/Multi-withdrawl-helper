币安提币助手 - 配置说明
======================

配置文件位置
----------
本程序的配置文件存储在用户文档目录下的"币安提币助手"文件夹中：
%USERPROFILE%\Documents\币安提币助手\config.ini

这样设计是为了解决以下问题：
1. 避免在程序目录写入配置文件时的权限问题
2. 确保打包后的程序也能正常保存配置
3. 方便用户找到和备份配置信息

首次运行
------
第一次运行程序时，会自动在以上位置创建配置文件。
您需要在设置中填写API密钥和其他配置信息。

注意事项
------
- 请勿手动编辑配置文件，以免格式错误
- 如需重置所有设置，只需删除上述目录中的config.ini文件
- 配置文件中包含API密钥等敏感信息，请妥善保管
